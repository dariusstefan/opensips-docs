Using OpenSIPS and SylkServer together one can provide the following services:

 * [Multimedia multiparty conferencing](http://sylkserver.com/applications.phtml) for SIP end-points (MSRP switch, RTP mixer, Conference Info event package)
 * [SIP / XMPP gateway](http://sylkserver.ag-projects.com/projects/sylkserver/wiki/DesignXMPP) for MSRP chat, Presence and wide-band audio (Jingle)
